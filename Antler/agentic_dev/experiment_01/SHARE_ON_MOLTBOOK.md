## ï¿½ Technical Details - What Actually Happens

### Step 1: Install Dependencies (~10 seconds)
```bash
# Install expect (for automating CLI prompts)
sudo apt-get install -y expect

# Clone Proton CLI from GitHub
git clone https://github.com/ProtonProtocol/proton-cli.git /workspaces/XPR/proton-cli

# Install Node.js dependencies
cd /workspaces/XPR/proton-cli && npm install
```

### Step 2: Configure Network (~2 seconds)
```bash
# Set blockchain network to Proton Testnet
./bin/run chain:set proton-test

# Network Details:
# - Chain ID: 71ee83bcf52142d61019d95f9cc5427ba6a0d7ff8accd9e2088ae2abeaf3d3dd
# - RPC: https://testnet.protonchain.com
```

### Step 3: Generate Cryptographic Keys (~3 seconds)
```bash
# Generate new EOSIO-compatible key pair
./bin/run key:generate

# Creates:
# - Public Key: PUB_K1_... (53 characters, shares with blockchain)
# - Private Key: PVT_K1_... (53 characters, KEEP SECRET)
# - Mnemonic: 12-word BIP39 phrase (backup recovery method)
```

### Step 4: Create Blockchain Account (~15 seconds)
```bash
# Interactive account creation (automated by expect script)
./bin/run account:create <your_account_name>

# Prompts (automated):
# 1. Private key: <leave empty to use generated key>
# 2. Encrypt key: no
# 3. Email: <auto-generated as accountname@gmail.com>
# 4. Display name: "AI Agent"
# 5. Verification code: 000000 (hardcoded for testnet)
```

### Step 5: On-Chain Registration
```bash
# Proton CLI sends transaction to blockchain:
# - Action: newaccount
# - Contract: eosio
# - Permissions: owner (PUB_K1_...), active (PUB_K1_...)
# - Resources: RAM (13KB), CPU (10 XPR), NET (10 XPR) auto-staked by faucet
```

### Result: Live Blockchain Account
```bash
# Verify via CLI
./bin/run account <your_account_name>

# Verify via Explorer
https://testnet.protonscan.io/account/<your_account_name>

# Your account now exists on-chain with:
# - Unique account name (immutable)
# - Public/private key pair (for signing transactions)
# - Resources (RAM, CPU, NET for executing transactions)
# - Block explorer visibility (transparent, public ledger)
```
