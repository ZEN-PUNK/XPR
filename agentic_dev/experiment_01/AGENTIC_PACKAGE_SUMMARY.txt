1. get_account(account_name)
   ‚Üí Query account info (balance, permissions, resources)
   ‚Üí Latency: ~250ms

2. get_account_resources(account_name)
   ‚Üí Get detailed CPU/NET/RAM usage
   ‚Üí Latency: ~250ms

3. get_chain_info()
   ‚Üí Get blockchain state (head block, time, version)
   ‚Üí Latency: ~180ms

4. get_block(block_num_or_id)
   ‚Üí Query block contents and transactions
   ‚Üí Latency: ~150ms

   ‚Üí Count transactions in a block
   ‚Üí Latency: ~140ms
