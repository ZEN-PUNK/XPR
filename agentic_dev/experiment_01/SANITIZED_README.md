# Experiment 01 - Sanitized

⚠️ **IMPORTANT: All sensitive information has been removed**

This experiment folder has been sanitized to remove Azure credentials:
- Azure subscription IDs → `AZURE-SUBSCRIPTION-ID`
- Resource group names → `AZURE-RESOURCE-GROUP`
- Function app names → `AZURE-FUNCTION-APP-NAME`
- Deployment endpoints → `https://YOUR-FUNCTION-APP.azurewebsites.net`

## Original Purpose

This experiment contains the initial MCP server implementation with FastMCP on Azure Functions.

## To Use This Code

Replace all placeholders with your actual Azure resource values:
1. `AZURE-SUBSCRIPTION-ID` → Your Azure subscription ID
2. `AZURE-RESOURCE-GROUP` → Your resource group name
3. `AZURE-FUNCTION-APP-NAME` → Your function app name
4. `YOUR-FUNCTION-APP` → Your function app hostname
